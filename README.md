# 🏋️ Kinetic Armour - Gym Website

A modern and responsive gym website built using HTML, CSS, and JavaScript. This project provides a complete fitness website UI with multiple sections and online payment integration.

---

## 🚀 Features

- Fully Responsive Design 📱
- Modern UI/UX Design 🎨
- Gym Sections:
  - Home
  - About
  - Services
  - Classes
  - Schedule
  - Pricing
  - Contact
- Smooth Scrolling Navigation
- Razorpay Payment Integration 💳
- Join Form with User Details

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Razorpay API
