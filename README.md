# Kinetic Armour Gym Website

A responsive gym website built using HTML, CSS, and JavaScript. This project includes multiple sections such as Home, About, Services, Classes, Schedule, Pricing, Contact, and a Join form with Razorpay payment integration.

## Features

- Responsive gym website design
- Smooth navigation
- About, Services, Classes, Schedule, Pricing, and Contact sections
- Join form for users
- Razorpay payment integration
- Modern fitness website layout

## Technologies Used

- HTML5
- CSS3
- JavaScript
